About the System

The BATANGAS Dengue Forecast System is a web-based application designed to help monitor and predict dengue outbreaks in the BATANGAS.

This system provides visual data through maps and dashboards to support early awareness and decision-making for communities and local authorities.

By using historical data and forecasting techniques, the system aims to reduce the impact of dengue through timely information and preventive action.
